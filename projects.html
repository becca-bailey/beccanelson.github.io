BrightFolio 

This is my Dev Bootcamp final project which my team and I completed in one week. Our goal was to create a visual organization system for images, documents, and files built on Ruby on Rails. BrightFolio uses Amazon Web Services S3 for file storage, and our focus was on UI features such as easy navigation and interactive front-end javascript.

Free Parking 

This was a Dev Bootcamp client project built on Ruby on Rails. My primary contribution to this project was an email invite system built with Action Mailer which allows users to invite a new or existing user to join a group using an automatically-generated secret token.
